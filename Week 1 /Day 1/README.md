# SQL Query Practice Collection – Day 1 Training

A beginner-friendly SQL practice repository created as part of **Day 1 Database Training**.  
This project contains basic to intermediate SQL queries using an Employee Management Database.

---

#  Project Overview

This repository is designed to help students and beginners practice SQL concepts through hands-on examples.

The project covers:
- Basic Queries
- String Matching Queries
- Date Queries
- Aggregate Functions
- Group By Queries
- Having Clause Queries

---

#  Topics Covered

##  Basic Queries
- Selecting all records
- Selecting specific columns
- Filtering records using conditions

##  String Matching Queries
- Using `LIKE`
- Pattern matching
- Character filtering

##  Date Queries
- Filtering using dates
- Year and month extraction
- Recent records

##  Aggregate Queries
- `SUM()`
- `AVG()`
- `MIN()`
- `MAX()`
- `COUNT()`

##  Group By Queries
- Department-wise analysis
- Salary grouping
- Employee statistics

##  Having Clause Queries
- Filtering grouped records
- Aggregate-based conditions
