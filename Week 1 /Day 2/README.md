#  Project Overview

Day 2 training is designed to improve SQL problem-solving skills using real-world style queries on an Employee Management Database.

This module covers advanced query writing techniques including:
- Sorting records
- Combining multiple tables
- Nested queries
- Aggregate comparisons
- Department-wise analysis
- Correlated subqueries
